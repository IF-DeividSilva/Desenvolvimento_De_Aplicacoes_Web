Instruções

Bem-vindo ao Quiz Educacional de Inglês! Este quiz foi desenvolvido para ajudar você a praticar e melhorar suas habilidades em inglês.

Você pode responder às perguntas usando sua voz. O sistema reconhecerá suas respostas e fornecerá feedback imediato.

Para começar, clique no botão "Iniciar Quiz". Você verá uma pergunta e poderá responder usando os comandos de voz disponíveis.

Após responder todas as perguntas, você verá seus resultados e feedback.

Comandos de voz disponíveis:

"yes", "no", "zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"

Comandos de voz para avançar e voltar questões: "right" (para avançar), "left" (para voltar)

Você pode usar os botões abaixo para iniciar o quiz, responder perguntas ou encerrar o quiz.

Use o botão "Reiniciar Quiz" para começar novamente.
